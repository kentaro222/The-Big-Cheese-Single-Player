# The-Big-Cheese-Single-Player
The digital game version of The Big Cheese.  Single-player only. Play against the CPU.

ビッグ・チーズ (The Big Cheese) ルール解説
『ビッグ・チーズ』は、部下（ネズミ）を使って仕事を競り落とし、効率よくプロジェクトを完了させて得点を競うゲームです。

1. ゲームの概要
目的: 仕事を完了させて得られる「チーズポイント」を集め、最初に40点に到達すること。
プレイ人数: 3~6人 （このデジタル版はプレイヤー1名、CPU3名で固定です）
プレイ時間: 約20分

3. 内容物
仕事カード: 36枚
数字の書かれた普通の仕事（2、4、6、8、10、12、20）が各4枚
特別な仕事（「ビッグ・チーズ(THE BIG CHEESE)」、「却下(VETO)」）が各4枚
部下ネズミ: 6色、各11個
得点トラックカード: 9枚（0から40まで）
サイコロ: 2面、4面、6面、8面、10面、12面、20面が各1個
※2面サイコロがない場合は、20面体で代用します（1〜10＝1点、11〜20＝2点）。

4. ゲームの準備
得点トラック: 「0」から「40」まで一列に並べます。
山札: 仕事カードをよく混ぜ、伏せて中央に置きます。
部下ネズミ: 各プレイヤーは自分の色のネズミを11匹受け取ります。
そのうち1匹を得点トラックの「0」に置きます。
残り10匹が、ゲーム開始時に手元にある部下になります。
親の決定: じゃんけんなどで最初の親を決めます。

5. ゲームの流れ
誰かが40点に達するまで以下の手順を繰り返します。

① 仕事の公開
親が山札からカードを1枚めくります。これが今回入札が行われる「仕事」です。

② 入札
親から時計回りに、その仕事に投入する「部下の数」を宣言します。
ルール: 前のプレイヤーよりも大きい数（競り値）を宣言するか、「パス」をします。「パス」をしたプレイヤーは、この回の入札が終わるまで数の宣言ができなくなります。
終了: 1人を除いて全員がパスしたら競り終了です。
落札: 最後に残ったプレイヤーがそのカードを獲得し、宣言した数と同数のネズミをカードの上に置きます（これを「途中の仕事」と呼びます）。

③ 時間の経過（ネズミの帰還）
誰かが仕事を落札した直後、以下の処理を行います。
今回落札された仕事以外の、すべての「途中の仕事」から部下を1匹ずつ持ち主の手元に戻します。
1人で複数の仕事を抱えている場合は、そのすべてから1匹ずつ戻ります。
※今回落札したばかりの仕事からは、まだ戻りません。

④ 仕事の完了と報酬
カードの上の部下が全員手元に戻った瞬間、その仕事は完了します。
サイコロを振る: カードに書かれた数字と同じ面数のサイコロを1回振ります。
得点: 出た目がそのまま得点（チーズポイント）になり、得点トラックのマーカーを進めます。
終了したカード: 得点処理後、カードは捨て札にします。
⑤ 親の交代
今回の仕事を競り落とした人が次の親になり、新しくカードをめくります。

6. 特別な仕事カード
数字が書かれていない特殊なカードが2種類あります。これらは完了しても点数は入りませんが、強力な能力として手元に置いておけます。
却下 (VETO)
自分の競りの順番が回ってきた時に使えます（パス中は不可）。
現在競りにかかっている仕事と、このカードをセットで捨て札にします。
直ちに次のカードをめくり、競りをやり直します。
ビッグ・チーズ (THE BIG CHEESE)
自分の仕事が完了し、サイコロを振った直後に使えます。
カードを捨て札にすることで、サイコロを1回振り直すことができます。

7. ゲームの終了
勝利条件: 誰かの得点が40点に届いたら即座に終了し、その人の勝ちです。
同時到達: 複数人が同時に40点を超えた場合は、点数が高い方の勝ちです。同点なら勝利を分け合います。
山札切れ: 山札がなくなったら捨て札を混ぜて新しい山札を作ります。
戦略のアドバイス:
大きな数字で落札すれば高い得点のチャンスがありますが、部下が手元に戻るまで時間がかかり、その間他の競りに参加できなくなります。手元の部下の数と、ゴールまでの点数計算が重要です。
